"# 3Estates" 


## Instructions to run the project locally

  
### Prerequisites

Install the following prerequisites

[NPM](https://nodejs.org)

### Clone the project
```bash
  git clone https://github.com/rk-rishikesh/Brandvilla.git
```
### Install node modules
```bash
  npm install
```

### To deploy this project run

```bash
  npm run start
```
